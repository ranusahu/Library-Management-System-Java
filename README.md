# Library Management System (Java)

This is a simple Java console-based Library Management System application.

## Features
- Add new books
- Display all books
- Search a book by title
- Remove a book

## Concepts Used
- ArrayList
- Classes and Objects
- Loops and Conditions
- Scanner class

## How to Run
1. Compile the program:
   javac LibraryManagementSystem.java
2. Run the program:
   java LibraryManagementSystem
